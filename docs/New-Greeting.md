# New-Greeting

## SYNOPSIS
Greeting people.

## SYNTAX

```
New-Greeting [-Name] <String> [-Greeting <String>] [-FooBar <String>] [<CommonParameters>]
```

## DESCRIPTION
Greet people at a conference.

## EXAMPLES

### Example 1
```powershell
PS C:\> New-Greeting -Name 'PSConfEU'
```

```
Hello PSConfEU
```

### Example 2
```powershell
PS C:\> 'a'..'c' | New-Greeting
```

```
Hello a
Hello b
Hello c
```

Greet a, b, c.
